# 3D Developer Portfolio

Root directory of the 3D Portfolio project. Contains configuration files and the main source directory.

## Project Overview
This is a modern, interactive 3D portfolio website built to showcase developer skills and projects. It leverages the power of:
- **React**: For building the user interface.
- **Three.js (@react-three/fiber)**: For rendering immersive 3D graphics and models.
- **Tailwind CSS**: For responsive and modern styling.
- **Vite**: For fast development and build tooling.

## Key Features
- **3D Hero Section**: Interactive 3D computer model.
- **About Section**: Overview of skills and services.
- **Experience**: Timeline of work experience.
- **Tech Stack**: Interactive balls representing technologies.
- **Projects**: Grid display of portfolio projects.
- **Testimonials**: Feedback from clients/colleagues.
- **Contact Form**: 3D interactive earth with email integration.
- **Admin Panel**: For managing content dynamically.

## Getting Started
To run this project locally:

1.  **Install Dependencies**
    `ash
    npm install
    `

2.  **Start Development Server**
    `ash
    npm run dev
    `

3.  **Build for Production**
    `ash
    npm run build
    `
