# canvas

Three.js canvas components for 3D, including models and environments.

## Folder Contents

| File / Folder | Description |
| :--- | :--- |
| **Avatar.jsx** | 3D component rendering the developer's avatar. |
| **Ball.jsx** | 3D interactive ball component used in the Tech section. |
| **index.js** | Contains all constant data for the app (nav links, service descriptions, experiences, projects, testimonials). |
| **Planet.jsx** | 3D model component for the Contact section (e.g., Earth or Planet). |
| **Stars.jsx** | Background component rendering a star field effect. |


## Contribution
This folder contributes to the specific functionality described above. It ensures modularity and separation of concerns within the project architecture.
