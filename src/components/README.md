# components

Reusable React components that make up the UI of the application.

## Folder Contents

| File / Folder | Description |
| :--- | :--- |
| **canvas** | Directory containing Three.js 3D canvas components. |
| **About.jsx** | Section describing the developer's background and skills. |
| **Contact.jsx** | Contact form section with a 3D Earth model. |
| **Experience.jsx** | Timeline section showcasing professional work experience. |
| **Feedbacks.jsx** | Testimonials section showing client or colleague feedback. |
| **Footer.jsx** | Footer component for the bottom of the page. |
| **Hero.jsx** | Component for the 3D Hero section, featuring the immersive intro. |
| **index.js** | Contains all constant data for the app (nav links, service descriptions, experiences, projects, testimonials). |
| **Loader.jsx** | Loading spinner component shown while 3D models are loading. |
| **MobileNotice.jsx** | Component displaying a notice or alternative view for mobile devices. |
| **Navbar.jsx** | Navigation bar component providing links to different sections. |
| **SocialLinks.jsx** | Component displaying social media links. |
| **Tech.jsx** | Section displaying the technology stack using 3D interactive balls. |
| **Works.jsx** | Portfolio section displaying projects in a grid layout. |


## Contribution
This folder contributes to the specific functionality described above. It ensures modularity and separation of concerns within the project architecture.
