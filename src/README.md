# src

Source code directory containing all React components, assets, and logic.

## Folder Contents

| File / Folder | Description |
| :--- | :--- |
| **assets** | Contains static assets like images and icons. |
| **components** | Contains React components. |
| **constants** | Directory containing related files. |
| **context** | Directory containing related files. |
| **data** | Directory containing related files. |
| **hoc** | Directory containing related files. |
| **utils** | Directory containing related files. |
| **index.css** | Global CSS styles, including Tailwind directives and base styles. |
| **main.jsx** | The entry point of the React application, rendering the App component into the DOM. |
| **styles.js** | Common JavaScript styles or constants used across components. |


## Contribution
This folder contributes to the specific functionality described above. It ensures modularity and separation of concerns within the project architecture.
