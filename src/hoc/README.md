# hoc

Higher-Order Components (HOCs) used for wrapping components with shared logic (e.g., section wrappers).

## Folder Contents

| File / Folder | Description |
| :--- | :--- |
| **index.js** | Contains all constant data for the app (nav links, service descriptions, experiences, projects, testimonials). |
| **SectionWrapper.jsx** | Higher-Order Component that wraps sections with Framer Motion animations and consistent styling. |


## Contribution
This folder contributes to the specific functionality described above. It ensures modularity and separation of concerns within the project architecture.
